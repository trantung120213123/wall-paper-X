# Code of Conduct

## Our Commitment

We want this project to be open, respectful, and productive for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus on technical discussion and evidence.
- Assume good intent and ask for clarification when needed.
- Accept feedback professionally.

## Unacceptable Behavior

- Harassment, hate speech, or personal attacks.
- Doxxing or sharing private information.
- Repeated disruptive behavior in issues or pull requests.

## Enforcement

Project maintainers may edit, hide, lock, or remove content that violates this policy.
Severe or repeated violations may result in a ban from the project spaces.

## Reporting Conduct Issues

If you encounter conduct violations, report to project maintainers privately with:

- What happened
- Where it happened (issue/PR/discussion)
- Links/screenshots (if available)
