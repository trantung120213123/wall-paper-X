# Support

## Before You Open an Issue

1. Read `README.md`.
2. Confirm build prerequisites are installed.
3. Reproduce on a clean build (`build-msvc` removed).
4. Check existing issues.

## Include This Information

- Windows version
- Visual Studio / MSVC version
- Exact command you ran
- Full error output
- Steps to reproduce

## Fast Triage Tip

If the problem is installer-related, include:

- Whether UAC prompt appeared
- Install path used
- Last status line shown in installer UI
